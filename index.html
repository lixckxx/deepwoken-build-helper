      Import from Deepwoken Builder

      Step-by-Step Instructions:

      1. Copy your Deepwoken Builder URL or ID (e.g., abc123a)
      2. Paste it in the box below and click "Open API Link"
      3. A new tab will open showing JSON data
      4. Select ALL the text on that page (Ctrl+A or Cmd+A)
      5. Copy it (Ctrl+C or Cmd+C)
      6. Come back to this tab and paste it in the text area below
      7. Click "Import Build"
      Tip: The JSON data should look like: {"stats":{"strength":50,...},"talents":[...]}
